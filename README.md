# Godot Minigame 1
## Devlog
A node is like an egg, while a scene is like a carton. Each egg can be moved individually and can be removed from the carton. However, the entire carton could be moved to move all the eggs simultaneously. Like this, a scene could be used to interact with multiple nodes simultaneously.
